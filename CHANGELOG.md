# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - Unreleased

### Added

- StackForge-generated TypeScript CLI package baseline.
- `scan`, `explain`, and `policy` commands.
- JSON and simple YAML manifest parsing.
- Capability heuristics for shell, filesystem, browser, network, messaging, secrets, database, and package-manager surfaces.
- Markdown and JSON report rendering.
- Deterministic approval-plan gates for critical, high, medium, low, and policy-denied findings.
- Local allow/warn/deny policy files.
- Fixtures, unit tests, CLI smoke tests, approval-gate coverage, and validation script integration.
